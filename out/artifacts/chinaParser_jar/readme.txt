Как пользоваться парсером
1. Скачать Java
https://www.java.com/ru/download/
2. В файл links.txt добавить (разделяя симфолом переноса строки (нажать Enter)) ссылки на товары, с которых хотите парсить контент.
3. Запустить парсер: 
	Для Windows:
	3.1. Перейти в каталог с парсером и запустить командную строку.
	3.2. В командной строке написать 
			java -jar chinaParser.jar полный_адрес_папки_куда_сохранять_без_кавычек
			Пример:
			java -jar chinaParser.jar D:\
	Для Linux и MacOS:
	3.4. Запустить терминал
	3.5. Перейти cd имя_каталога с парсером
	3.6. В командной строке написать 
			java -jar chinaParser.jar полный_адрес_папки_куда_сохранять_без_кавычек
			Пример:
			java -jar chinaParser.jar /Users/user/Desktop/fuck_the_China/
4. В случае, если парсер выдает ??????? вместо китайских иероглифов, установить поддержку языка.
